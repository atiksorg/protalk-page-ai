# ProTalk Page AI

Chrome-расширение для анализа веб-страниц и умного заполнения форм с помощью ИИ через ProTalk API.

## Возможности

- **Анализ страницы** — извлекает текст после финального рендера, включая SPA/React/Vue-приложения
- **Вопросы к ИИ** — задавайте вопросы по содержимому любой страницы
- **Input Assistant** — иконка ✨ появляется рядом с полями ввода и помогает их заполнить
- **Заполнение форм** — AI анализирует контекст страницы и заполняет одно поле или всю форму целиком
- **Улучшение текста** — рерайт уже введённого текста: исправление ошибок, профессиональный тон
- **Несколько моделей** — Grok, GPT, Claude, Gemini на выбор
- **Статистика токенов** — отслеживание общего расхода

## Установка

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `protalk_page_ai`

## Использование

### Первая настройка

1. Кликните на иконку расширения в панели инструментов
2. Введите credentials:
   - **Email** — ваш email в ProTalk
   - **Bot ID** — идентификатор бота
   - **Bot Token** — токен бота
3. Нажмите **Сохранить**

### Вопросы по странице

1. Откройте любую веб-страницу
2. Кликните на иконку расширения
3. Введите вопрос в поле **Ваш вопрос**
4. Нажмите **Задать вопрос**

Расширение дождётся финального рендера страницы и передаст актуальный текст в ИИ.

### Input Assistant

Включается в **Настройках** → переключатель **Input Assistant**.

После включения на каждом текстовом поле при фокусе появляется иконка ✨. По клику открывается мини-меню:

| Действие | Когда доступно |
|---|---|
| 💡 Заполнить поле | Всегда |
| ✏️ Улучшить текст | Если поле уже заполнено |
| 📋 Заполнить всю форму | Если поле находится внутри `<form>` |

**Input Assistant не активируется** на полях `password`, платёжных данных (`cc-*`, `cvv`, `card`, `ssn`, `cvc`).

### Настройки

Нажмите ⚙️ в правом верхнем углу попапа:

- **Модель ИИ** — выбор модели для всех запросов
- **Input Assistant** — включение/выключение помощника на всех сайтах
- **Статистика** — общий расход токенов
- **Выйти из профиля** — очистить сохранённые credentials

## Модели

| Идентификатор | Описание |
|---|---|
| `x-ai/grok-4.1-fast` | По умолчанию, быстрый |
| `openai/gpt-5-mini` | GPT, баланс скорости и качества |
| `anthropic/claude-haiku-4.5` | Claude, хорош для текста |
| `google/gemini-3-flash-preview` | Gemini, быстрый |

## Структура файлов

```
protalk_page_ai/
├── manifest.json       # Конфигурация расширения (MV3)
├── background.js       # Service worker: API-запросы, счётчик токенов
├── content.js          # Content script: захват текста, Input Assistant
├── popup/
│   ├── popup.html      # UI расширения
│   ├── popup.js        # Логика попапа
│   └── popup.css       # Стили
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

## Permissions

- `storage` — хранение credentials и настроек
- `activeTab` — доступ к активной вкладке
- `scripting` — инжекция content script
- `host_permissions: <all_urls>` — необходимо для Input Assistant на любых сайтах

## API

Расширение использует ProTalk API как прокси к OpenRouter:

- **Endpoint:** `https://ai.pro-talk.ru/api/router`
- **Upstream:** `https://openrouter.ai/api/v1/chat/completions`
- **Auth:** `Bearer {botId}_{botToken}`
