# ProTalk Page AI

Chrome-расширение для анализа текста веб-страницы с помощью ИИ через ProTalk API.

## Возможности

- **Анализ текста страницы**: Автоматически извлекает текст из текущей вкладки
- **ИИ-ассистент**: Задавайте вопросы по содержимому страницы
- **Несколько моделей ИИ**: Выбор между Grok, GPT, Claude, Gemini
- **Статистика токенов**: Отслеживание расхода токенов
- **Сохранение настроек**: Credentials хранятся локально

## Установка

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `protalk_page_ai`
5. Расширение появится в панели инструментов

## Использование

### Первая настройка

1. Кликните на иконку расширения
2. Введите ваши credentials:
   - **Email**: ваш email в ProTalk
   - **Bot ID**: идентификатор бота
   - **Bot Token**: токен бота
3. Нажмите **Сохранить**

### Запрос к ИИ

1. Откройте любую веб-страницу
2. Кликните на иконку расширения
3. Введите ваш вопрос в поле **Ваш вопрос**
4. Нажмите **Задать вопрос**
5. Получите ответ от ИИ на основе текста страницы

### Настройки

Нажмите на шестеренку (⚙️) в правом верхнем углу для доступа к настройкам:

- **Модель ИИ**: Выберите предпочитаемую модель
  - `x-ai/grok-4.1-fast` (по умолчанию)
  - `openai/gpt-5-mini`
  - `anthropic/claude-haiku-4.5`
  - `google/gemini-3-flash-preview`
- **Статистика**: Просмотр расхода токенов
- **Выйти из профиля**: Очистить сохраненные credentials

## Permissions

- `storage` — хранение credentials и настроек
- `activeTab` — доступ к текущей вкладке
- `scripting` — инъекция content script

## API

Расширение использует ProTalk API:
- Endpoint: `https://ai.pro-talk.ru/api/router`
- Модели: OpenRouter (Grok, GPT, Claude, Gemini)
